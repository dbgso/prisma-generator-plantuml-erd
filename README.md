# prisma-generator-plantuml-erd

> This generator was bootstraped using [create-prisma-generator](https://github.com/YassinEldeeb/create-prisma-generator)

## Development

This project uses pnpm for package management.

### Prerequisites

- Node.js 24.11.0 (managed by Volta)
- pnpm 8.x

### Setup

```bash
# Install pnpm if not already installed
npm install -g pnpm

# Install dependencies
pnpm install

# Run tests
cd packages/generator
pnpm run test

# Build
pnpm run build
```

### Security

This project maintains zero vulnerabilities by using pnpm's superior dependency override capabilities:

```bash
# Check for vulnerabilities
pnpm audit

# Expected output: "No known vulnerabilities found"
```
